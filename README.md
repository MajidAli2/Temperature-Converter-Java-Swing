# Temperature Converter (Java Swing)

A simple graphical application to convert temperatures between Celsius, Fahrenheit, and Kelvin scales, built using Java Swing.

## Features

- User-friendly GUI developed with Java Swing
- Convert temperature values between Celsius, Fahrenheit, and Kelvin
- Real-time input and display of results
- Input validation with helpful error messages
- Light-weight and easy to run on any Java-supported platform

## How It Works

1. Enter the temperature value in the text field.
2. Select the units to convert **from** and **to** (e.g., Celsius to Fahrenheit).
3. Click the "Convert" button to see the result instantly.



## Requirements

- Java 8 or higher

## How to Run

1. Clone the repository:
    ```sh
    git clone https://github.com/yourusername/Temperature-Converter-Java-Swing.git
    ```
2. Compile the program:
    ```sh
    javac TemperatureConverter.java
    ```
3. Run the program:
    ```sh
    java TemperatureConverter
    ```
